# Caffeine Calculator

My take on a caffeine calculator; a web app built with React, TypeScript, and Tailwind CSS.

The calculation engine uses a multiplicative model informed by peer-reviewed literature, factoring in species caffeine density, roast degradation, brew efficiency, and brewing conditions to give you a solid estimate of caffeine per cup and total caffeine in your brew.

Also includes a brand reference tab with caffeine data for popular coffee chains and a methodology tab that explains the science behind the numbers. All units can be toggled between metric and imperial.
